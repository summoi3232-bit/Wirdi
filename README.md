# وِردي — تطبيق متابعة الورد الأسبوعي 🌙

تطبيق ويب بسيط (ملف واحد HTML/CSS/JS بدون أي إعداد بناء) لمتابعة ورد قرآني أسبوعي من السبت إلى الجمعة.

## المزايا
- توزيع أسبوعي: السبت→الحج، الأحد→القصص، الإثنين→يس، الثلاثاء→الزخرف، الأربعاء→المجادلة، الخميس→الناس، والجمعة عداد للإكثار من الصلاة على النبي ﷺ.
- علامة إنجاز/عدم إنجاز لكل سورة + الوقت التقديري للقراءة.
- حفظ تلقائي فوري في متصفحك (localStorage) — لا حاجة لأي خادم أو تسجيل دخول.
- نسخة احتياطية: تنزيل/استيراد ملف JSON لكل بياناتك.
- تقارير أسبوعية وشهرية وسنوية قابلة للطباعة مباشرة من المتصفح (Ctrl+P).

## تشغيل محلي
افتح ملف `index.html` مباشرة في أي متصفح، لا يحتاج تثبيت أي شيء.

## الرفع على GitHub
```bash
git init
git add .
git commit -m "وِردي - نسخة أولى"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO_NAME.git
git push -u origin main
```

## النشر على Netlify
**الطريقة الأولى (سحب وإفلات):**
1. افتح https://app.netlify.com/drop
2. اسحب مجلد المشروع كاملاً (أو فقط ملف index.html) وأفلته في الصفحة.
3. سيصدر لك Netlify رابطًا مباشرًا للموقع خلال ثوانٍ.

**الطريقة الثانية (ربط مستودع GitHub):**
1. من لوحة Netlify: New site from Git → اختر مستودع GitHub الذي رفعت إليه المشروع.
2. اترك إعدادات البناء فارغة (لا يوجد build command، ومجلد النشر هو الجذر `/`).
3. اضغط Deploy site.

> ملاحظة: بما أن التطبيق يحفظ البيانات في متصفح الجهاز (localStorage)، فإن كل جهاز/متصفح يحتفظ بنسخته الخاصة من التقدم. استخدم زر "نسخة احتياطية" بشكل دوري، خصوصًا إذا كنت ستستخدم التطبيق من أكثر من جهاز.
